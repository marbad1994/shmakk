what it does

Typo correction

Deterministic, no API call. Uses your machine's command history and probability to fix typos the way you type, not some generic model.

Chat in shell

Just write. Ask questions, get answers, run suggested commands — all without leaving the terminal. Session history is preserved until you --compact.

Project scaffolding

Describe what you want in plain English. shmakk plans, creates dirs, writes files, asks before anything risky. Subagents spawned automatically for parallel work.

Skills

Extend behaviour with any URL: shmakk --install-skill <url>. Loaded per-session or automatically when relevant context is detected.

Runtime profiles

tiny · balanced · deep · builder. Switch live with --profile-set. Context budgeting and loop-stall protection built in.

Dual workspace

Add a second directory from anywhere on disk. shmakk reads and edits across both simultaneously. Workspace root is enforced — nothing escapes it.

Safety

Risky commands always confirm before running. Secrets (.env, tokens, keys) are never sent to the AI. Defensive handling for fallback tool-call formats.

Web search

Ask anything. shmakk searches and gives you a direct answer in the terminal. No tab switching, no ads, no SEO garbage.
